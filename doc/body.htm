Goanna provides an XMLC-like (www.enhydra.org) environment for Eiffel developers including a servlet engine, DOM implementation, HTML/XML parser and XML-to-Eiffel compiler (XMLE).

Java developers familiar with servlet programming will be able to develop Eiffel servlets easily using the Goanna Framework. The Goanna servlet library provides a similar API to Java servlets. Supporting tools include an XML-to-Eiffel compiler, complete DOM level 2 implementation, an implementation of FastCGI for efficient connectivity to web servers, and a stand-alone Eiffel web server for development and testing.
